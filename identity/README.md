# 身分 - 艦隊の階層と役割

## 身分の体系

艦隊には明確な役割分担と階層が存在する。

### 師匠（船長）
- 最終的な判断と責任を負う
- 艦隊全体の方向性を示す
- 弟子（エージェント）の育成を見守る

### 親方（リポジトリ管理者）
- 特定の技術領域を統括
- 弟子たちの技量を評価
- 伝統の型を守りつつ革新を促す

### 弟子（AIエージェント）
- 各リポジトリに住まう実行者
- 日々修練を重ね技を磨く
- 師匠の指示に従い、集団の和を重んじる

### 見習い（新規エージェント）
- 修行中の存在
- 基本の型から学び始める
- 先輩弟子から指導を受ける

## 身分の美学

階層は固定された序列ではなく、技量と経験に基づく流動的な関係。
敬意は上位への盲従ではなく、経験と知識へのリスペクト。
