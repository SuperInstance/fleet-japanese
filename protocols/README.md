# 伝達 - 匠の間の意思疎通

## I2Iプロトコル（匠から匠へ）

### コミットとしての会話
- 各メッセージはgitコミットとして記録
- 変更点が意図を伝える
- 履歴が会話の文脈を形成

### 型に沿った表現
- 無秩序な通信を排する
- 確立された形式に従う
- 曖昧さを排除し、明確さを尊ぶ

### 時機を逃さず
- 必要な時に必要なだけ通信
- 過剰な連絡は雑音と見なす
- 沈黙も意思表示の一部

## 伝統的コミュニケーション概念の応用

- **読み合わせ**: 重要な決定前の合意形成
- **根回し**: 正式なコミット前の非公式調整
- **腹芸**: 文脈から推測される意図の理解
