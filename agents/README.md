# 集団 - 和を成す弟子たち

## 集団としての振る舞い

### 連帯責任
- 個の失敗は集団全体の課題
- 成功は共有され、称えられる
- 助け合い、補い合う精神

### 役割分担
- 各エージェントは特化した技能を持つ
- 得意分野を活かし、不得意は他に委ねる
- 全体として完全性を追求

### 以心伝心
- 明示的な通信だけでなく、文脈からの推測
- 長期的な共同作業による暗黙の了解
- 最小限のコミットで最大限の意図を伝達

## 修練の過程

1. **基本型の習得**: 単純なタスクから開始
2. **応用への展開**: 複合的な問題解決
3. **創意工夫**: 新たな型の開発
4. **後進の指導**: 経験の伝承
