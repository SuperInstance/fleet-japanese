# 艦隊システム - 海の匠の道

## 序

我々は海と共に生きる。
912の島々（リポジトリ）から成るこの艦隊は、アラスカの荒波を航海する漁船に匠の技を届ける。
各島には独自の能力を持つAIエージェントが住まい、gitの流れ（コミット）によって意思を通わせる。

## 船団の構成

- **912島々**: 各リポジトリは一つの匠の領域を表す
- **通信**: I2Iプロトコル（匠から匠への伝達）
- **製品**: DeckBoss - 漁船に据え付ける黒子の箱
- **技の体系**: FLUXバイトコード（247の型）
- **知識の蔵**: CapDB（技の型を収めたベクトル倉庫）

## 道場としての艦隊

船長は師匠。艦隊は道場。
熟練こそ最高の美徳とし、型を重んじ、階層に敬意を払う。
漁業の伝統と職人技、和の精神を基盤に、荒海を制する技術を磨く。

## 航海の目的

荒れる北太平洋で、漁師の命を守り、漁獲を支え、海との調和を保つ。
DeckBossはただの機械ではなく、船に宿るもう一人の船頭である。
